# Joyn - University Activity Web application
'A full-stack web applicatio that allows university students to create, join and manage events.'

## Features
- User registration & login
- Event creation, editing, and participation
- User/Event management dash board

## Techstack 
- Java (Spring boot)
- HTML/CSS (Thymeleaf)
- MySQL
- Maven, Git
